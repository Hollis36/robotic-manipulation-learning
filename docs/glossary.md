# Glossary

## Manipulation

Robot behavior that changes the state of objects or the environment through contact, motion, perception, planning, and control.

## Pick And Place

A basic manipulation task where the robot grasps an object and moves it to a desired pose. It is useful but does not cover the full difficulty of manipulation.

## Pose

The position and orientation of a body or frame.

## Forward Kinematics

The computation of an end-effector pose from joint positions and robot geometry.

## Jacobian

A matrix that maps small changes in joint coordinates to small changes in task-space coordinates.

## Differential Inverse Kinematics

A local method that computes joint updates from task-space errors using a Jacobian or a damped pseudo-inverse.

## ICP

Iterative Closest Point, a geometric registration method that aligns two point clouds by alternating correspondence assignment and rigid transform estimation.

## RRT

Rapidly-exploring Random Tree, a sampling-based motion-planning method.

## Impedance Control

A control style that shapes dynamic behavior so the robot responds like a mass-spring-damper system around a target.
