## How To Use This Index

1. Start with the local chapter notes and casebook.
2. Check the official course notes when a concept is unclear.
3. Use Drake tutorials when moving a concept case into `drake_labs/`.
4. Use broader resource lists only after the course stack is clear; otherwise it is easy to collect papers without building skill.
