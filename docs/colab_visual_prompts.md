# Colab Visual Prompts

The current Colab concept images are generated by `tools/generate_colab_svg_illustrations.py`.
That script writes reviewable SVG sources under `book/assets/colab/svg/` and exports the
notebook-ready 1600x900 PNG files under `book/assets/colab/`.

The prompts below are retained as art-direction references for future redesigns. They define
the same professional robotics education illustration style, while the SVG generator is the
canonical asset pipeline. The images explain concepts; numerical evidence comes from the
code-generated result figures.
