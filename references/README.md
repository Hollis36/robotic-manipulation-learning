# References

The chapter PDFs in the workspace are local study material. They are intentionally ignored by Git so this repository can be published as a learner-owned project without republishing the original notes.

Recommended workflow:

1. Keep the PDFs locally beside this repository or in a private reference folder.
2. Write summaries in `chapters/*/notes.md` using my own words.
3. Use `CITATION.md` when referencing the original course notes.
4. Add links to public course pages instead of copying long passages.
