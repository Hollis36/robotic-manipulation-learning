BROWSER JupyterLite 无需服务器，适合矩阵、几何、规划和控制的小实验。
CLOUD NOTEBOOK Google Colab 打开公开仓库 notebook，首个 setup cell 自动 clone 源码。
CLOUD IDE GitHub Codespaces 完整 Python 环境，可运行 pytest、casebook 和静态站点构建。

Codespaces 编译运行

  1. 打开 Codespaces 后等待 devcontainer 自动安装依赖。
  2. 在 VS Code 任务面板运行 `RML: Run All Casebook Examples`。
  3. 修改 notebook 或 `src/rml` 后运行 `RML: Verify Repository`。
