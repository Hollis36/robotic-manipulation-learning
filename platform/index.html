ROBOTIC MANIPULATION LEARNING

在线学习平台

一套部署在 GitHub Pages 的机器人操作学习平台：先读课程图谱，再进入浏览器内 JupyterLite 编程，最后用 GitHub Codespaces 做完整工程实验。

READ 课程学习书 打开 Jupyter Book，按章节学习目标、检查点和练习任务推进。 CODE JupyterLite 在线编程 无需服务器，在浏览器中运行 Python notebook，适合概念实验和小案例。 BUILD GitHub Codespaces 启动云端 VS Code，安装完整依赖，运行测试、构建网页和调试代码。

推荐学习流

  1. 先在课程学习书里读 `00 Manipulation Stack`。
  2. 进入 JupyterLite 修改 notebook 参数并运行。
  3. 需要完整 Python 环境时打开 Codespaces。
  4. 把学习记录提交回 GitHub 仓库。
