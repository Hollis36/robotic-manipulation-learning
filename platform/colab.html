GOOGLE COLAB NOTEBOOKS

Colab 学习入口

用 Google Colab 打开公开仓库中的 notebook。每个 notebook 的首个 setup cell 会自动 clone 仓库并加载 `src/rml`，适合快速运行矩阵、几何、规划、控制和强化学习小实验。

FAST NOTEBOOK Google Colab 适合云端 Python notebook、参数修改和结果记录。
STATIC BROWSER JupyterLite 无需登录，在浏览器里运行轻量实验。
FULL IDE Codespaces 完整 VS Code 环境，可运行测试、casebook 和站点构建。

Colab 使用流

  1. 打开一个 notebook 后先运行第一个 setup cell。
  2. 完整运行一次示例，确认输出和章节概念对应。
  3. 按 Practice Task 修改参数，比较结果变化。
  4. 需要运行测试或提交代码时切换到 Codespaces。
