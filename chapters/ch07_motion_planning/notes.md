# Chapter 7 Notes

## 本章核心问题

当环境更复杂、手写末端轨迹不再可靠时，机器人如何自动找到可行运动路径？

## 关键概念

- IK 解决目标约束，motion planning 解决路径可行性。
- RRT 等采样方法通过随机探索构建可达路径。
- 轨迹优化可以把平滑性、碰撞和动力学约束纳入目标。

## 公式与图示

- RRT 的基本循环：sample -> nearest -> steer -> collision check -> add node。

## 实现想法

- 在二维平面上实现 RRT，障碍物用圆表示。
- 检查路径每条边是否穿过障碍物。
