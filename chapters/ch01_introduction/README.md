# Chapter 1: Introduction

## Focus

Manipulation is not just pick-and-place. This chapter frames robot manipulation as an integration problem across perception, planning, control, contact, uncertainty, and open-world task reasoning.

## Learning Outputs

- Build a taxonomy of manipulation problems.
- Define personal learning goals for the rest of the repository.
- Use Chapter 1 as the mental model for why later chapters are split into hardware, geometry, perception, planning, control, and learning.
