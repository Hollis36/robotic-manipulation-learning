## 我的回答

### 1. Manipulation 和 Pick-and-place 的区别

Pick-and-place 是 manipulation 的一个常见子任务：目标通常是把物体从 A 放到 B。Manipulation 更宽：它关注机器人如何通过接触改变物体或环境状态，包括推、拉、插、拧、折叠、整理、滑动、利用环境约束、失败恢复等。

### 2. 五个日常操作任务

| 任务 | 主要困难 |
| --- | --- |
| 把水槽里的盘子放进洗碗机 | 盘子堆叠、遮挡、滑动接触、狭窄空间规划。 |
| 折叠毛巾 | 软体物体状态难感知，动作会改变形状。 |
| 打开抽屉取物 | 需要识别把手、控制拉动方向，并处理关节约束。 |
| 把插头插进插座 | 需要高精度位姿估计、接触引导和柔顺控制。 |
| 从杂物箱中抓取目标物 | 遮挡、未知物体、抓取候选多、失败后需要重试。 |

### 3. 课程结束希望做出的项目

做一个“桌面整理”小系统：输入桌面 RGB-D 或模拟点云，识别可抓取物体，规划抓取和放置位置，执行一个简化 pick-and-place 或 push-and-place 策略，并能记录失败原因。
