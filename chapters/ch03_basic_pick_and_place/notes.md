# Chapter 3 Notes

## 本章核心问题

如果先假设物体位姿已知，机器人如何把“抓取并放置”拆成坐标变换、运动学和轨迹问题？

## 关键概念

- 坐标系命名和变换方向必须一致，否则 bug 很隐蔽。
- 正运动学从关节角计算末端位姿。
- Jacobian 把关节速度映射到任务空间速度。
- 微分逆运动学是局部方法，需要处理奇异和步长。

## 公式与图示

- 平面机械臂末端位置：

```text
x = sum_i l_i cos(q_1 + ... + q_i)
y = sum_i l_i sin(q_1 + ... + q_i)
```

- 阻尼最小二乘：

```text
dq = J^T (J J^T + lambda^2 I)^-1 e
```

## 实现想法

- 用 NumPy 实现 2D 齐次变换并测试组合顺序。
- 写一个两连杆或三连杆平面机械臂。
- 用目标点误差迭代求 IK。
